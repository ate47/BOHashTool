Tools

Search

The search tool allows you to search for a particular hash, it will try to search for permutation of you string.

You can set a prefix using the '|' keyword, it won't do any permutation the element before it. If you add a second one it will act a suffix and won't do any permutation after it. If you end your hash by ".gsc" or by ".csc", it will search for the hash with from the last '/' to the '.' and to the same hash with ".csc" or ".gsc" at the end.


Large search

The large search tool allows you to search for multiple hashes at the same time, unlike Search, it won't do any permutation


Large hash

The large hash tool allows you to hash multiple elements at the same time with the FNV-1a method, it won't search for the hash in your files. It can be useful to create a list of hashes from components.


Replace

The replace tool allows you to replace all the found elements in your directory, the syntax is the same as the one returned by the Search tools.


Extractor

The extractor tool allows you to extract all the elements in your directory, it will write it into an output directory.
